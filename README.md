# 早押しクイズチャレンジ

シンプルな Node.js ベースのクイズアプリです。PC / スマートフォン双方からプレイでき、プレイヤー名とスコア、タイムでランキングを作成します。画面上にはアクセス用の QR コードが表示されます。

## セットアップ

Node.js (>= 18) がインストールされている環境で、以下の手順で起動できます。

```bash
npm install  # 依存パッケージはありませんが、初回に lockfile を生成したい場合は実行してください。
npm start
```

サーバーは `http://localhost:3000` で起動します。

## 機能

- 早押しクイズ形式（10問、複数選択式）
- 回答時間の計測と記録
- ランキングの閲覧および更新（最高記録のみを保存）
- QR コード生成による簡易アクセス共有
- モバイルフレンドリーな UI
